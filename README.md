# GitBucket MCP Server

A Model Context Protocol (MCP) server for GitBucket API, mirroring GitHub MCP tool signatures.

## Overview

This MCP server provides AI agents (like OpenCode Desktop, Claude Desktop) with tools to interact with GitBucket instances using the same tool signatures as GitHub MCP.
