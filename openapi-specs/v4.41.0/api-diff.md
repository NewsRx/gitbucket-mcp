The release notes mention "Keyword search for issues and pull requests." This is a **Web UI feature**, not an API change:
- Located in: `IssuesController.scala`, `PullRequestsController.scala`
- Redirects simple keyword queries to the web search endpoint
- Does NOT add new API parameters or endpoints
- Existing API query parameters continue to work as before
